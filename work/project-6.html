Overview

Scan-Verse is a new-generation content-recording platform that generates realistic 3D models from everyday objects and restores memories in 360°. It solves not just the problem of "how to preserve" an object, but the deeper human need to make memories tangible, shareable, and interactive — turning a scanned teacup into a living story you can gesture open.

3D Scanning Object Digital Preservation Gesture Interaction
Background

Three converging trends show that users want to own and interact with their memories digitally — but existing tools create friction that stops them before they start.

Rise of Digital Collecting

67% vs 13%

>67% of Gen Z and Millennials have recorded important memories digitally, yet only 13% have tried 3D scanning — blocked by high tool barriers and unclear use cases. (Statista 2023)

Personalized Digital Spaces

>72% want more

Over 72% want to personalize their digital spaces — avatars, virtual rooms, collection walls. Demand for "self-curation" and "interactive memory" is rising fast. (Adobe 2022)

AR/VR Interaction Preference

57% find it more immersive

57% of AR users find interacting with objects more immersive than viewing displays. Gesture-based input is one of the most natural methods for personalized emotional experiences. (XR Association 2023)

Capture & Scan Process

The capture process uses mobile photos or videos. The user sets the central axis; a rotating guide sphere captures the object from all angles — video keeps ambient sound, photo is silent.

STEP 1

Find a valuable object

Something meaningful — an heirloom, a childhood toy, a gift.

STEP 2

Capture it

Photo or video using the rotating guide sphere for complete coverage.

STEP 3

Generate the model

System identifies the object, generates a clean 3D mesh, and stores it in the app.

STEP 4

Interact in the album

View, rotate, gesture, and share scanned items from the 3D gallery.

Gesture Interaction

Gesture responses are customizable per model type. Humanoid/character models (e.g., a sculpture) and object models (e.g., a teacup or chair) each respond to different gesture vocabularies — making the interaction feel tailored to the object's nature.

Interaction Flow

Tap → Zoom → Rotate → Effect

Basic model view → tap object → zoom in/out → finger rotate → trying effects. Plus a customization phase and tutorial.

Gesture Recognition

Thumbs-up / Yeah / OK

Custom gestures recognized for selecting object effects and visual filters. Models trained via Teachable Machine — image samples → classes → exported model + JavaScript integration.

Real Example

The Teacup Memory

A user scans his grandfather's old teacup with Polycam. Gesture interaction opens the lid, raises steam, and triggers a recorded voice and story — bringing back the memory of sitting together over tea.

Platform Experience
3D Immersive Museum

Explorable space

Scanned objects displayed in an immersive, walkable virtual museum — curated by the user.

Device Interface

App + tablet

Welcome → get started → upload photos → generate model → gallery → settings. Designed for both phone and tablet.

QR Sharing

Physical + digital

Generate a QR code → export (photo album / share link / print as offline exhibition entry) → others access the memory space via personal page or family sharing.

Summary and Reflections
Before

"Memories sit in flat photo albums you rarely open."

After

"Memories become 3D objects you can rotate, gesture, and share — things you can almost touch."

The project revealed that preservation and interaction are two different problems — and most tools only solve the first. The key design insight was making the 3D model the starting point, not the end point: once it exists, gesture interaction, family sharing, and the 3D museum become natural extensions. Scan-Verse is an interactive AR digital-memory platform that lets users collect, share, and interact with personal memories in an immersive way — turning preserved objects into living, shareable experiences.
