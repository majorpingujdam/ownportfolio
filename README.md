# Portfolio

Personal portfolio site — UX & HCI design student.

Plain HTML/CSS/JS. No build step. Open `index.html` in a browser, or host on GitHub Pages.

## Edit checklist

Things to change before sharing:

- `index.html` — replace `Your Name`, `YN`, `hello@yourname.com`, social links.
- `index.html` — swap the four `.project` blocks for real work (title, year, tag, chips).
- `style.css` — `--accent: #c9461c` is the single brand color; change it once to retheme everything.

## Deploy (GitHub Pages)

1. Push to GitHub.
2. Repo → Settings → Pages → Source: `main` branch, `/` root.
3. Done — site is at `https://<username>.github.io/<repo>/`.
